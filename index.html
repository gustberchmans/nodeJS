API Documentatie

Users CRUD

  • GET /api/users - Haal alle users op
    • Query params: limit, offset (optioneel)
  • GET /api/users/:id - Haal één user op via ID
  • POST /api/users - Maak een nieuwe user
    • Body (JSON): { "name": "Bob", "email": "bob@example.com", "age": 25 }
  • PUT /api/users/:id - Update een bestaande user
    • Body (JSON) met velden die je wilt updaten
  • DELETE /api/users/:id - Verwijder een user

Users Search

  • GET /api/users/search - Zoek users op name, email of age
    • Query params:
      • name - (string) partial match, case-insensitive
      • email - (string) partial match, case-insensitive
      • age - (number)
      • limit - (number, default 10)
      • offset - (number, default 0)
      • sortBy - veld om op te sorteren (default "name")
      • sortOrder - "asc" of "desc" (default "asc")
    • Voorbeeld: /api/users/search?name=bo&age=25&limit=5&offset=0&sortBy=age&sortOrder=desc

Allergens CRUD

  • GET /api/allergens - Haal alle allergens op
    • Query params: limit, offset, sortBy, sortOrder
  • GET /api/allergens/:id - Haal één allergen op via ID
  • POST /api/allergens - Maak een nieuwe allergen
    • Body (JSON): { "name": "Pinda", "description": "Kan allergische reacties veroorzaken" }
  • PUT /api/allergens/:id - Update een bestaande allergen
    • Body (JSON) met velden die je wilt updaten
  • DELETE /api/allergens/:id - Verwijder een allergen

Allergens Search

  • GET /api/allergens/search - Zoek allergens op name of description
    • Query params:
      • name - (string) partial match, case-insensitive
      • description - (string) partial match, case-insensitive
      • limit - (number, default 10)
      • offset - (number, default 0)
      • sortBy - veld om op te sorteren (default "name")
      • sortOrder - "asc" of "desc" (default "asc")
    • Voorbeeld: /api/allergens/search?name=pinda&limit=5&sortBy=name&sortOrder=asc
